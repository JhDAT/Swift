## Quenstion

 다음 튜플 연산에 대한 결과는?
 
 - ("1", 1) < ("3.14", 99)
 
 - ("일", 1) > ("이", 2.0)
 
 - (1, "zebra") < ("2", "apple")
 
 - ("blue", -1) < ("purple", 1)

 - ("blue", false) < ("purple", true)


<br>
<br>
<br>

## Answers

- ("1", 1) < ("3.14", 99) - True

- ("일", 1) > ("이", 2.0) - True

- (1, "zebra") < ("2", "apple") - 오류 다른 Type은 비교 할수 없습니다.

- ("blue", -1) < ("purple", 1) - True

- ("blue", false) < ("purple", true) - 오류 bool값은 비교를 할수가 없습니다.
