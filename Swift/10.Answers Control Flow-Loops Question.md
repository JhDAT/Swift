<br>

## Quenstion

3. for문을 이용하여 3의 10 제곱에 대한 결과값을 표현하는 코드구현 
2. for문을 이용하여 10부터 1까지 내림차순으로 출력하는 코드 구현

<br>
<br>
<br>

## Answers

1)
```
let three = 3
let range = 10
var sum = 1


for _ in 1...range {
    sum *= three
}
print(sum) // 59049
```
2)
```
for num in (1...10).reversed() {
    print(num)
}```
